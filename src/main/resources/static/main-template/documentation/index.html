Installation - top

Follow the steps below to get started with your Site Template:

  • Open the Main-files/HTML Folder to find all the Templates Files
  • You will need to Upload these files to your Web Server using FTP in order to use it on your Website.
  • Make sure you upload the required files/folders listed below:
    • HTML/css - Extra Stylesheets Folder
    • HTML/images- Images Folder
    • HTML/js - Javacripts Folder
    • HTML/css/main.css - Main Stylesheet File you can also rename it to style.css (for wordpress)
    • HTML/index.html - Index File/Homepage
    The other files can be used according to your preferences.
  • You're now good to go..! Start adding your Content and show off your Brand New Beautiful Website in style.

How To Use CSS / JS Files - top

CSS and JS both are important part of the website. we've included all the plugins css/js inside main.css, and main.js files with proper commenting to avoid more http request to server and to make website farster. main.css Inside HTML > CSS you can find main.css which contains all the css.

	<link rel="stylesheet" href="css/main.css">

Bootstrap.css Inside HTML > CSS you can find bootstrap.css which is the main part of the website as Doclist based on bootstrap 3 it is recommended to include bootstrap.css file right after main.css file.

	<link rel="stylesheet" href="css/bootstrap.css">

main.js Inside HTML > JS you can find main.js which contains all the custom and plugins code used in website.

	<script src="js/main.js"></script>

Color Schemes - Top

You can change your Website's Color Scheme in an instant. You simply need to change the Color Code in the css/main.css file & you are good to go.

Colors are defined with :root in css/main.css you can change the root value to change the primary and secondary colors 

	<head>
		...
		<link rel="stylesheet" href="css/main.css" type="text/css" />

	</head>

Now simply change the Codes according to your requirements

Changing Fonts - top

Change your Fonts on the Fly as we have included fonts in main.css file. Doclist uses 2 Fonts namely: Poppins and Open+Sans from the Google Fonts Library. You can find the Linking to the Font Files in the main.css file

	@import url(https://fonts.googleapis.com/css?family=Open+Sans:400,
600|Poppins:300,400,500,600,700);

Google Maps - top

You can add Embedded Google Maps to any Page using the following setup:

Step 1:

Add the Google Maps Specific Scripts in the <main.js> Tag below the js/main.js script linking:

	<script src="https://maps.google.com/maps/api/js?key=YOUR-API-KEY"></script>

Step 2:

Add a div which will contain your Google Map:

	<div id="dc-locationmap" class="dc-locationmap"></div>
